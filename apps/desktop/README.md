# LoanLife Edge Desktop (Electron + React)

Desktop-first UI for loan digital twins, AI risk predictions, ESG compliance tracking, and blockchain auditability.

## Scripts
- npm run dev — start Vite and Electron for development
- npm run build:react — build React UI
- npm run build — build UI and package Electron app
- npm start — start Electron (expects built files)

## Structure
Electron code in `electron/`, React UI in `src/`, entry in `index.html`.
